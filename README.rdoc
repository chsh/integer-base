= integer-base

* http://github.com/kachick/integer-base
* https://rubygems.org/gems/integer-base
* http://rubyforge.org/projects/integer-base/

== DESCRIPTION

use free-format in positional numeral system

== FEATURES

* easy build upper 36 basenumbers
* use original syntax (0 1 2 -> 0 a b, etc)

== SYNOPSIS

* setup
    require 'integer/base'

* extend
    require 'integer/base/import'

* to_i
    my_chars = %w[0 a]
    'a'.to_i(my_chars) #=> 1
    'a0'.to_i(my_chars) #=> 2

    my_chars = ['0', *'A'..'I']
    'a'.to_i(my_chars) #=> 1
    'a0'.to_i(my_chars) #=> 10
    'aib'.to_i(my_chars) #=> 192

* to_s
    192.to_s(my_chars) #=> 'aib'

== REQUIREMENTS

* Ruby 1.9.2 and later (tested 1.9.2, 1.9.3)

== INSTALL

* sudo gem install integer-base
